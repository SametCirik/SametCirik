## 自己紹介

私は**型にはまらない方法**で解決策を見つけることに情熱を注ぐテクノロジー愛好家です。常に失敗から学び、自分自身を前進させています。

アイデアを形にし、プロトタイプを作成するために、AI支援開発（「vibe coding」）を頻繁に活用しています。これは強力な学習方法ですが、現在の私の目標は、これらのツールへの依存を減らし、より自立して作業できるよう基礎スキルを強化することです。

私にとってコーディングは単なるツールではなく、独自のソリューションとビジョンをゼロから構築する芸術です。現在、**ゲーム開発（C# & Unity）**および**デスクトップUI/UXアプリケーション（Java）**の分野で活動しています。

私が最も情熱を注ぎ、高いポテンシャルを秘めていると信じているプロジェクトをご覧ください。***私が書くすべてのコードは、学びと進歩の物語です。***
