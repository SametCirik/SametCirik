## 关于我

我是一名技术爱好者，专注于通过**非传统方法**寻找解决方案，并不断从错误中学习以推动自己前进。

我经常利用人工智能辅助开发（“vibe coding”）来制作原型并将想法变为现实。虽然这是一种强大的学习方式，但我目前的重点是减少对这些工具的依赖，并加强我的核心基础技能，以便更独立地工作。

对我来说，编码不仅仅是一个工具；它是从零开始构建我自己的解决方案和愿景的艺术。我目前活跃于**游戏开发（C# & Unity）**和**桌面UI/UX应用（Java）**领域。

来看看那些最让我兴奋、并且我认为具有高潜力的项目。***我写的每一行代码都是一个学习和进步的故事。***
